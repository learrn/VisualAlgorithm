# VisualAlgorithm
### 项目介绍
该项目为了实现一个Android版本[visualgo](https://visualgo.net)，
visualgo是由Steven Halim博士在2011年发布的一款可视化学习算法的工具，用于帮助其学生更好地理解数据结构和算法。   
通过动画与代码的结合使得用户更容易理解算法的运作和思想。   
目前该项目仅完成了排序模块的编写，其他模块将在未来继续完善。
